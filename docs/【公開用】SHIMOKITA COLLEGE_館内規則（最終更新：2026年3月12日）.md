2026.3.12時点版

# **はじめに** {#はじめに}

この館内規則にはSHIMOKITA COLLEGEに入学する人が守るべき施設の利用ルールが記されています。館内規則は大きく分けて下記の5つの観点から記載されています。カレッジ生（SHIMOKITA COLLEGEに入学した人の総称、以下同じ。）全員が本規則を十分に理解し、遵守してください。な  
お、高校生については、「SHIMOKITA COLLEGE 高校生の生活上のルール」も館内規則に含まれます。

* 安全上の観点  
* 衛生上の観点  
* 美観維持の観点  
* 近隣住民との関係性維持の観点  
* その他集団生活を円滑に送るのに必要な観点

1. # **リラックス食堂の利用について** {#リラックス食堂の利用について}

   SHIMOKITA COLLEGEでは、食堂で食事を取ることをレジデンシャル教育の一環と位置づけており、食堂で共に食事を取ることで生まれる交流や対話からの学びを重視しています。そのため、SHIMOKITA COLLEGE入居者（以下、カレッジ生）には賃貸借契約に付随して、食堂の利用も必須となっています。食堂のルールは別途食堂よりアナウンスをします。ルールを守って食堂を利用してください。  
   * 食堂の営業日  
     　土日祝日を除く平日です。ゴールデンウィーク、お盆期間、年末年始に合計10日の休業日があります。また、イベント時には特別営業が行われる場合があります。休業日やイベントの詳細は、年度ごとに作成する食堂営業カレンダーを確認してください。  
   * 食堂の営業時間  
     1. 朝食　7:00から10:00まで（早出し営業として6:30から簡易的なメニューで提供）  
     2. 夕食　19:00から21:00まで（夕食は21:45まで取り置き対応）  
        ※早出しや取り置きのルールは、別途食堂よりアナウンスをします。ルールが遵守されない場合は、予告なく中止することがあります。  
   * 長期不在におけるフードロス対策について  
     1. 不在者の連絡は食堂に共有され、また不在が多い期間などは仕入れの調整を行い、フードロス等が発生しないようにしていますので、あらかじめの不在連絡にご協力お願いします。  
   * 留意事項  
     　食堂の喫食を停止することはできません。 また、長期不在による一時的な喫食停止と返金も行っておりません。（詳細は賃貸借契約を確認してください。）アレルギーや信仰上の理由、体調面などでご懸念がある場合は、ご相談ください。

2. # **電気・水道・ガス、通信の利用について** {#電気・水道・ガス、通信の利用について}

   電気、水、通信等の費用はみなさんが節約して使っていただく前提で学費に含まれています。カレッジ生の故意・不注意により、個室および共用部の水道光熱費が高騰した場合には、カレッジ生個人への請求ないしはカレッジ生全員に対して月額料金の増額・値上げを検討する可能性もありますので注意してください。特に仮想通貨等のマイニング等の行為により許容範囲を逸脱する電気使用があった場合は、個人に対して追加費用の請求をします。

3. # **個室の利用について** {#個室の利用について}

   個室や個室に備え付けられている備品や備付家具などは以下の注意事項を守り、大切に使用してください。  
* 個室内の窓には各自で用意したカーテンを必ず設置してください。（２人部屋は予め備え付けのカーテンを使ってください。）  
* 個室のテラスは避難用になります。テラスへ続く避難用ドアは非常時以外は絶対に開けないでください。避難用ドアのサムターンカバーに破損があった場合、修理費5,500円(税込)をお支払いいただきます。  
* 自転車 （折り畳み自転車含む）を個室に持ち込まないでください。  
* 個室の設備、設置してある家具、備品等の使用に際して、 万が一損傷、故障や不具合が発生した場合は運営事務局に直ちに報告してください。  
* 故意又は過失により個室の設備、設置してある家具、備品等が損傷、滅失した場合にはその修繕費用等を負担していただきます。  
* 退去の際に誤って個室の備え付けの家具・家電・備品等を持ち出した場合、同等品の購入費用をお支払いいただきます。  
* 個室の電球等の消耗品は個人の費用負担で交換してください。

4. # **共用部の利用について** {#共用部の利用について}

   個室以外はカレッジ生が共に使用する共用部です。共用部を利用する場合は以下の点に注意してください。  
* 共用部に私物を放置しないでください。万が一共用部に私物が放置されていた場合は定期的に撤去され、1週間経過後に通告無く廃棄されます。  
* 掲示物も運営事務局によって許可された場所以外への掲示はしないでください。  
* 共有スペースの設備、備品の改造行為はしないでください。  
* 平日休日に関わらず、日-金の間の7:30-15:00の間、共用部に清掃が入ります。清掃エリアは1-5Fの共有エリア、2,3,5Fのキッチン、1,2,3Fのバスルームとなっています。それぞれ清掃時間でも利用ができます。日頃から共有スペースを清潔に利用し、清掃業者の方々に感謝と挨拶を心がけましょう。

**各スペースを利用する際の注意事項**

* **キッチン**  
  * 調理器具・食器などは、使用した者が使用後速やかに自ら洗浄し、所定の場所に収納してください。  
  * 冷蔵庫内の保管物で管理者にて不衛生と判断した場合、通告無く破棄する場合があります。  
  * キッチンでの火災発生には十分注意し、調理中にその場を絶対に離れないでください。  
* **暖炉**  
  * ガス式暖炉のため取り扱いに十分に注意し近くに可燃物を置かないでください。  
  * 使用した人が責任をもって消してください。  
* **テラス**  
  * **テラスは9:00 \- 21:00使用可能です。**ただし天候などの諸事情により閉鎖することがあります。

  * ## テラスのグループでの利用は数名を限度とします。

  * ## イベントなどで使用したい場合は、事前に運営事務局の許可を取ること。その際は利用する責任者と人数を申請してください。

  * ## テラスで洗濯物を干さないでください。

  * 火気厳禁です。テラスの防水膜を溶かした場合、建物全体の水漏れの原因になります。火気を使用した個人に多額（数百万円程度）の修繕費が請求される恐れがありますので、絶対に火気を使わないでください。

5. # **共有備品について** {#共有備品について}

* カレッジ生が個人で持っているもの以外のものは、共有の備品としてカレッジ生全員で利用するものです。丁寧に扱い、使用後は直ちに所定の場所に戻してください。  
* 故意又は過失により貸出備品等に損傷、滅失した場合にはその回復費用等を負担していただきます。  
* 退去の際に誤って共有の家具・家電・備品等を持ち出した場合、同等品の購入費用をお支払いいただきます。

6. # **駐輪場の利用について** {#駐輪場の利用について}

* カレッジ生ひとりに対して最大1台の駐輪まで可能です（折り畳み自転車やスクーターなども1台としてカウントします）。  
* 駐輪スペースには枠の上限があるため、上限枠を超える駐輪スペース利用希望があった場合は抽選を行い、利用者を決定します。  
* 駐輪する自転車には、番号が割り当てられ、駐輪シールが支給されます。運営事務局から支給された駐輪シールは見える場所に必ず貼ってください。  
* 駐輪場はカレッジ生及びカレッジ関係者に限り利用可能で、割り当てられた駐輪場のみ使用可能です。ゲストが自転車を駐輪できるスペースはないので、近隣の駐輪スペースを利用してください。  
* シールを貼っていない自転車、あるいは、割り当てられた番号とは違う番号の場所に置かれている自転車については無断駐輪とみなされます。  
* 施錠を必ず行い自らの責任で管理するものとし、盗難や破損等については運営事務局は一切の責任を負いません。  
* 無断駐輪の自転車は、運営事務局が許可なく撤去・処分する場合があります。

7. # **駐車場の利用について** {#駐車場の利用について}

* 駐車場は以下の場合に限り一時的に利用することができます。駐車場を利用する場合は事前に運営事務局にその旨を伝達して、許可を得てください。  
  * 身体的な困難等を有する方の利用  
  * 施設管理などに必要な業者の利用  
  * 小田急電鉄、UDS、HLABの来客などカレッジ関係者の利用  
  * カレッジ生の荷物の運搬などの一時的利用（事前に事務局の許可を取った上で、9時〜21時の間のみ30分以内の利用とする。入退去時などの引越し業者などついては、事前に調整し、所要時間のみの利用とする。）  
  * 緊急時（救急車両を含む）の利用  
* 空いているスペースは、避難経路に当たります。運営事務局の許可なく物を放置したり、ゲストの自転車、バイク、自動車を駐車したりしないでください。

8. # **玄関利用について** {#玄関利用について}

* 施設内の出入りは原則正面玄関を使ってください。メールボックス側の裏口を利用する際は周囲の近隣のご自宅と距離が近いため、時間帯を問わず配慮をし、声を出すことは慎しんでください。  
* 玄関口へのスケボー・自転車の乗り入れ禁止です。

9. # **ゲストに関して** {#ゲストに関して}

   このカレッジでは安全上の観点から、カレッジ生及びカレッジ関係者が招いたゲスト（以下「ゲスト」という。）に対して下記の通り一定の入館条件を設けております。万が一問題が発生した場合は、当該ゲストを招いたカレッジ生に責任が及びますので、ご注意ください。  
* ゲストの滞在可能時間は7:00 \- 24:00までとします。  
* 東京都青少年の健全な育成に関する条例により23時 \- 翌朝4時までの18歳未満の外出は原則禁止されております。18歳未満のゲストの滞在可能時間は7:00 \- 22:00までとします。  
* ゲストの滞在可能エリアは1・2Fのパブリックスペースのみで、個室のある占有スペースには内見する場合やその他運営事務局が特別に許可した場合を除いて立ち入りできません。  
* ゲストに鍵を渡し、カレッジ生の同行なしにゲストが単独で入館することはできません。単独入場に際しては、警察に通報する場合があります。  
* カレッジ生本人以外は宿泊することはできません。  
* ゲストが内見を希望する場合のみ、以下を守ったうえ、個室のある占有スペースへ立ち入りが可能です。内見を利用する場合は事前に運営事務局にその旨を伝達して、許可を得てください。  
* 内見対応の際に、個室にゲストを招き入れることはできません。

11. # **管理上必要な場合の個室内への立ち入りについて** {#管理上必要な場合の個室内への立ち入りについて}

* 火災・地震等の震災時や事件・事故等の緊急時等必要と認められる場合には、カレッジ生が不在であっても、運営事務局が個室に許可なく入室することがあります。  
* 消防設備の法定点検や個室設備の修理等管理上、個室に立ち入る必要がある場合は、事前にカレッジ生に告知し立ち入ることがあります。  
* カレッジ生は、法律に定められた重要な設備の点検については、特別な理由がない限り、運営事務局に協力してください。  
* 客観的事実に基づきカレッジ生が本規則に違反していると推定される場合、カレッジ生に連絡の上、運営事務局担当者が個室に立ち入ることがあります。

12. #  **郵便物・宅配便について** {#郵便物・宅配便について}

* 宅配便、書留郵便に関しては原則カレッジ生本人が直接受け取れる時間帯を指定して対面にて受領してください。  
* 運営事務局が個人宛の宅配便や書留郵便を代理で受け取ることはしません。  
* 本人が受け取れない時は予め自己責任のもと置配指定を行ってください。置配指定した荷物が紛失した場合は、運営事務局は責任を負いません。  
* 1Fに設置されている宅配ボックスは運営用なので利用できません。

13. # **ごみ出しについて** {#ごみ出しについて}

* ごみ出しの際は以下のルールを守ってください  
  * トラッシュエリアに掲示されている[世田谷区のごみ分別のルール](https://www.city.setagaya.lg.jp/02241/387.html)を確認し、分別してください。  
  * 粗大ごみ等の処分に費用が発生するごみは、排出するカレッジ生個人が費用を負担してください。  
* 悪質なルール違反者には、監視カメラの録画から個人を特定して対応します。

14. # **落とし物の取り扱いについて** {#落とし物の取り扱いについて}

* 貴重品（現金・財布・鍵など）については1週間程度事務局にて保管し、その後警察に移管します。  
* 貴重品以外については、1週間程度事務局にて保管し、その後処分します。

15. # **鍵について** {#鍵について}

* 紛失・破損時には、カレッジ生負担にて再発行となり下記費用が発生します。  
  * 共用部電子鍵（セコムキー）	：16,500円（税込）  
  * 個室物理鍵			：22,000円（税込）  
  * 机物理鍵(水回り付き部屋除く)	：5,500円（税込）  
* 鍵を再発行したのちに、紛失した鍵を発見しても再発行費用は返金しません。なお、発見した鍵は速やかに運営事務局に返却してください。

16. # **非常通報について** {#非常通報について}

* 不審者、火災その他命に関わる様な状況の際は直ちに、110番ないしは119番通報してください。  
* 漏電やガス漏れなどを検知して、警報が鳴ることがありますが、警備会社（セコム）と設備管理会社（小田急ビルサービス）が同時に駆け付けますので、到着をお待ちください。

17. # **飲酒について** {#飲酒について}

* 20歳未満の者の飲酒及び20歳未満の者の人に飲酒を勧誘、強制することは法律で禁じられています。飲酒を行った20歳未満の者及び勧誘・強制した者はそれぞれ強制退去処分となる可能性があります。  
* 個室および共用部で飲酒することは可能ですが、夜間は他のカレッジ生や近隣の方々に配慮してください。  
* カレッジの敷地内外を問わず、カレッジ周辺の屋外での飲酒は近隣の迷惑となりますので、厳に慎んでください。

18. # **喫煙について** {#喫煙について}

* カレッジの敷地内は館内外ともに全面禁煙です。（電子タバコや水タバコを含む）  
* 世田谷区全域が路上喫煙禁止地域のため、周辺の道路などでの喫煙は厳に慎んでください。

19. # **薬物について**

* 違法薬物、危険ドラッグ及びこれらに類似する物質を含む薬物や製品を所持、購入、譲受け、使用（適法な医療行為による場合を除く）または他のカレッジ生に供与しようとする行為は固く禁止します。発覚した場合、警察に通報の上、即時強制退去処分とします。  
* 処方薬（中枢神経抑制剤）及びこれに準ずる市販薬等の乱用（それを目的とする所持及び売買を含む）を禁止します。

20. # **騒音について** {#騒音について}

* 他のカレッジ生や近隣の住民の迷惑にならないよう騒音には十分注意してください。  
* 北側の個室について、22:00-翌8:00の間において、近隣の住民の迷惑になると想定されるような大きな音を出すことは慎んでください。  
* 近隣住民及び他のカレッジ生に騒音の被害を与え、繰り返しの注意にも従わない場合は強制退去処分となる可能性があります。

21. # **勧誘・販売について** {#勧誘・販売について}

以下の行為をカレッジ内で行うことは禁止です。

* マルチ商法、ネットワークビジネス等の勧誘・販売  
* 反復継続的にカレッジ内で物品やサービスを提供し、対価を得る行為  
* その他、違法・悪質な勧誘行為、カレッジ生の関係性に悪影響を及ぼすと運営事務局がみなした行為等

22. # **感染症対策について** {#感染症対策について}

* インフルエンザ・ノロウイルス・はしか等の感染力の強い病気にかかった場合は、カレッジ内の公衆衛生管理の観点から、運営事務局へ報告をお願いします。  
* 症状が出た場合には、マスク着用等他のカレッジ生への感染拡大防止に努めてください。

23. # **災害時の備えと対策について** {#災害時の備えと対策について}

* ​災害時に備え、予め避難経路を把握してください。  
* 避難通路確保のために廊下やエレベーターホール等共用スペースに私物を置かないでください。置かれていた場合は、撤去して処分することがあります。  
* 定期的に開催される防災訓練には、特別な事情がない限り必ず参加してください。

24. # **禁止事項について** {#禁止事項について}

    下記の禁止事項行為が発覚した場合は、直ちに強制退去処分となる可能性があります。なお、強制退去の場合であっても、契約書に定められている次の契約解除可能日又は契約期間満了日までの費用はお支払いいただきます。  
* 危険な火気の取扱い  
  * 敷地内での花火、爆竹、模造拳銃、バーベキュー、香煙、その他火器の使用及びガス、炭、ろうそくの使用、個室での喫煙  
* 危険物  
  * 武器や武器に準ずるような物を敷地内に持ち込むこと  
* 危険行為  
  * 本人及び他人に危害を及ぼすような行為や危険とみなされる行為  
* 暴力行為  
  * 本人及び他人に危害を及ぼす物理的および精神的な暴力行為  
* 様々なハラスメント行為 （パワハラ、セクハラ、いじめ等）[(ハラスメント防止に関するガイドライン](https://docs.google.com/document/d/1lkqO-owOjriBH2awIh2fK1Q9S1Wzeoj7layTdHIopHY/edit)参照)  
* 他人の誹謗中傷 （インターネット上での書き込みも含む）  
* 居室の改造  
  * 居室を無断で改造すること   
  * 改造行為にあたるか不明な場合は事前に必ず確認すること  
* ペット飼育・餌やり  
  * 敷地内に動物を持ち込むこと  
  * 敷地内に出入りする動物に餌を与えること  
* 賭博などの違法行為  
* 第19条（薬物）及び[別途定める薬物禁止規定](https://docs.google.com/document/d/1K5B9IAfer67fvVsNsRqzIwKfe7DY9CO_CAAThsxM-6s/edit)に該当する行為  
* 政治・宗教活動  
  * 敷地内で特定の政党を支持、若しくはこれに反対する政治活動、又は特定の宗教のための宗教活動やそれに係る目的のための集会を開催すること  
* 高校生に対し、「SHIMOKITA COLLEGE 高校生の生活上のルール」に抵触する行為、その他条例で青少年に禁止されている行為をさせたり手助けしたりする行為  
  * 高校生に午後11時～午前4時の外出をさせたり同行したりすること  
  * 高校生を自室に招き入れること

25. # **その他遵守事項について** {#その他遵守事項について}

26. # **館内規則に違反した場合の措置** {#館内規則に違反した場合の措置}

*  館内規則に違反した場合、1度目の違反では、当該カレッジ生本人への注意・指導及び関係するカレッジ生に対するヒアリングを行います。また、高校生及び大学生の場合は保護者等に指導を依頼します。  
* 注意・指導後も改善が見られない場合や2度目の違反があった場合、強制退去処分となる可能性があります。  
* 強制退去者は、在籍カレッジ生の同行があっても今後一切の入館を認めません。  
* 強制退去時の学費は、契約書に定められているとおりお支払いいただきます。  
* アラムナイ(カレッジ退去済み者)が館内規則に違反した場合、館内立ち入り禁止・slackやgoogleアカウントを含めたITツールの停止処分となる可能性があります。
